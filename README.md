# REST API Library Management with File Upload

This project is developed using Node.js and Express.js. It provides a RESTful API for managing books in a library and includes a file upload and management system with authentication.

## Features

### Library Management
- Add, update, and delete books
- Fetch books with pagination
- Filter books by author and genre
- Protected routes using JWT authentication

### File Management
- Upload files (images, PDFs, etc.)
- View list of uploaded files
- Download and delete files
- Access restricted using authentication

## Technologies Used
- Node.js
- Express.js
- MongoDB (Mongoose)
- JWT Authentication
- Multer
